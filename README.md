- ## 修改了一些快捷键
- 根据使用习惯，修改了一些快捷键：  
> 重复行或选中区域 ——> Ctrl+Shift+D  
> 开始新行 ——> Ctrl+Enter  
> 运行程序 ——> F5  
> 左侧显示 Project ——> Ctrl+1  
> 查找同一变量 ——> Ctrl+D
